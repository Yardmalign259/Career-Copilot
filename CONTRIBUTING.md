# Contributing to Career Copilot

First off — thank you! This tool is built for Indian freshers, by someone who's been there.

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR_USERNAME/Career-Copilot.git
cd Career-Copilot
```

### 2. Run Locally
- Open in VS Code
- Right click `index.html` → Open with Live Server
- Get a free Groq API key at console.groq.com

### 3. Make Your Changes
- One feature per PR
- Follow existing code style
- Add comments for complex logic

### 4. Submit a PR
- Clear title: `feat: add dark mode` or `fix: PDF upload on mobile`
- Describe what you changed and why

## Good First Issues
Look for issues labeled `good first issue` on GitHub.

## Code Style
- Vanilla JS — no frameworks
- Each file does ONE thing
- Comments in English (code) or Hinglish (user-facing strings)

## Questions?
Open a GitHub issue — happy to help.
